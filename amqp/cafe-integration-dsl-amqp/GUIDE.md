# Messaging with AMQP and RabbitMQ



## Run

```bash
docker-compose up // serves a RabbitMQ server instance
mvn spring-boot:run
```

## Test

```bash
mvn test  // run the sample test for Amqp/RabbitMQ
mvn verify -Pit // for functional tests
```
